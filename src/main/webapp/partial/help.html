Not logged in user

You are not logged in. If you only want to view a queue, that's cool, but to be able to join a queue, you will need to log in.

Viewing a queue

If you don't want to log in, you can open up any queue and see the virtual line of eagerly waiting people. This is done by clicking the name of the queue you want to look at in the Queues page.

Admin functions

Add an administrator

To add an administrator, go to the Admin page. Then enter the name of the user you wish add as administrator in the field under Admin administration and click Add. The user should now show up in the admin list. If not, please refresh the page.

Remove an administrator

To remove an administrator, go to the Admin page. Then, under Admin administration, click the remove button beside the name of the administrator you wish to remove. The user should now disappear from the admin list. If not, please refresh the page.

Note: There must always be at least one administrator. It is therefor not possible to remove an administrator if it is the last administrator.

Queue owner functions

Add a queue owner or moderator

To add a queue owner or moderator for a queue, first go to the Admin page. Then choose the queue you wish to add a queue owner or moderator for in the dropdown as below

After you have selected a queue, you enter the name of the user you wish to add either in the Owners or the Moderators input fileds, and then press the Add button. When that is done, the user has been added as either queue owner or moderator. If the list doesn't update automatically, please refresh the page and verify that the user has been added.

Remove a queue owner or moderator

To remove a queue owner or moderator from a queue, first go to the Admin page. Then choose the queue you wish to remove a queue owner or moderator for in the dropdown as when adding a queue owner. Click the red cross beside the user you want to remove, as in the picture below, and then confirm the removal in the popup that appears.

The priveleges of the user has now been removed. If the user doesn't update automatically, please refresh the page and verify that the user has been removed.

Hibernate a queue

Hibernating a queue means both clearing, locking and hiding the queue. This can be used for example when there will not be labs in a queue for a longer time.

To hibernate a queue, first go to that queues main page via the Queues page. Then go into the Manage queue dropdown, and select Hibernate. Verify in the popup that appears.

When you want to open the queue again, go into the same menu and select Unhibernate.

Queue moderator functions

Remove a queue position

To remove a queue position, click the red cross beside the name of the position you wish to remove.

Then select Remove twice to completely remove the queue position.

Note: Be careful. Once a position have been removed, it can note be restored.

Lock a queue

When a queue is locked, people can see the queue but can't join it. When locking a queue, people already in the queue will not be removed.

To lock a queue, first go to that queues main page via the Queues page. Then go into the Manage queue dropdown, and select Lock. Verify in the popup that appears.

To unlock a queue, do the same as above, but instead of choosing Lock, you choose Unlock.

Clear a queue

When a queue is cleared, all people in the queue will be removed. This can't be restored.

To clear a queue, first go to that queues main page via the Queues page. Then go into the Manage queue dropdown, and select Clear. Verify in the popup that appears.

Regular users

Queue status

The queues in QWait can have different status, which can be good to know. The status is indicated by the color and style of the queue name.

Style Icon Description
Active queue The queue is active and can be joined
Locked queue The queue is locked and can't be joined
Hibernated queue This queue is locked and hidden from the queue list, and can't be joined

Find a queue

The first step to queueing would be to find the queue you wish to join, right? So how is that done? Well, first go to the Queues page. There you can either scroll to find the queue you are searching for, or you could search for the queue. To scroll, just scroll - more info on scrolling will not be covered in this help section. Join a queue

To join a queue, you will first need to find the queue (see Find a queue). When you have found the queue, let's say you want to join the queue Allmänhandledning, click the queue name.

You now reach the queue page, where all the queueing takes place. If you are at a KTH computer, your location will be automatically obtained.

If your location is not automatically obtained, please type your location in the location field.

Now click the Join queue button, and you are done!

Note: In the queue list page, the boxes of the queues you are currently queueing in become blue.

Leave a queue

To leave a queue, you must first join a queue. If you haven't joined the queue you want to leave, we recommend you to not join that queue. If you have already joined the queue you want to leave, go to that queues main page and click the Leave queue button.
